The Movie API!

The Movie API, created by Sean Kopp-Reddy, allows you to search through it's extensive database of movies via movie title, director, genre, etc. Once you register you can add movies to your favorites list, or make a queue of movie's to watch in you "To Watch" list!

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of all movies /movies GET None A JSON object holding data about all movies.
Get info about a movie by title /movies/[title] GET None A JSON object holding info about a single movie; containing an id, title, description, genre info, director info, an image link, and info on whether it is currently a featured movie. Example:

{
"_id": "5f51af7e2de9e3bd754846d7",
"Title": "Fight Club",
"Description": "An insomniac office-worker and a devil-may-care soapmaker form an underground fight club that evolves into something much, much more.",
"Genre": {
"Name": "Drama",
"Description": "..."
},
"Director": {
"Name": "David Fincher",
"Bio": "...""
"Birth": "1962"
},
"ImagePath": "https://theposterdb.com/api/assets/70423/view?",
"Featured": true
}
Get a list of all genres /genres GET None A JSON object holding data about all genres (see below for format).
Get info about genre by name of genre /genres/[name] GET None A JSON object holding info about a single genre; containg an id, name, and description. Example:

{
"_id":"5f569b6a3f8444092e44449e",
"Name":"Fantasy",
"Description":"Fantasy films are films that belong to the fantasy genre with fantastic themes, usually magic, supernatural events, mythology, folklore, or exotic fantasy worlds. The genre is considered a form of speculative fiction alongside science fiction films and horror films, although the genres do overlap."
}
Get a list of all directors /directors GET None A JSON object holding data about all directors (see below for format).
Get info about a director by name /directors/[name] GET None A JSON object holding info about a single director; containing an id, name, and bio. Example:

{
"_id": "5f569cfc3f8444092e4444a1",
"Name": "Hayao Miyazaki",
"Bio": "Hayao Miyazaki is one of Japan's greatest animation directors. The entertaining plots, compelling characters, and breathtaking animation in his films have earned him international renown from critics as well as public recognition within Japan. The Walt Disney Company's commitment to introduce the films to the rest of the world will let more people appreciate the high-quality works he has given the movie-going public.",
"Birth": "1941"
}
Register as a new user /register POST A JSON object holding data about the new user; including username, password, email, and birthday- structured like:

{
"Username" : "sean",
"Password" : "password",
"Email" : "sean.koppreddy3@gmail.com",
"Birthday" : "1992-04-02"
}
A JSON object holding data about the new user, structured the same as input with userID.
Update user info /users/[name] PUT None A JSON object containing the newly updated information about the user, formatted as:

{
"_id": "5f51c1f4fe003fbfafb70c12",
"Username" : "sean",
"Password" : "password",
"Email" : "sean.koppreddy3@gmail.com",
"Birthday" : "1992-04-02"
}
Add movies to favorite list /users/[Username]/movies/[movieID] POST None A JSON object containing the newly updated information about the user; including FavoriteMovie array, formatted as:

{
"_id": "5f51c1f4fe003fbfafb70c12",
"Username" : "sean",
"Password" : "password",
"Email" : "sean.koppreddy3@gmail.com",
"Birthday" : "1992-04-02"
"FavoriteMovies": [
"5f51af7e2de9e3bd754846d7",
"5f51afd72de9e3bd754846d8",
"5f51b0262de9e3bd754846da"
],
}
Remove movies from favorite list /users/[Username]/movies/[movieID] DELETE None A JSON object containing the newly updated information about the user (see above row).
Deregister user /users/[Username] DELETE None A message verifying that your account has been deleted; "[user] has been deleted."
