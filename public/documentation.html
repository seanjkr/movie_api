The Movie API!

The Movie API, created by Sean Kopp-Reddy, allows you to search through it's extensive database of movies via movie title, director, genre, etc. Once you register you can add movies to your favorites list, or make a queue of movie's to watch in you "To Watch" list!

Business Logic URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of all movies /movies GET None A JSON object holding data about all movies.
Get info about a movie by title /movies/[title] GET None A JSON object holding info about a single movie, containing a title, director, and description. Example:
{
title: 'Fight Club',
director: 'David Fincher',
description: 'An insomniac office worker and a devil-may-care soapmaker form an underground fight club that evolves into something much, much more.'
}
Get info about genre by name/title of genre /genres/[name] GET None A JSON object holding info about a single genre, containg a genre name. Example:
{
name: 'Action'
}
Get info about a director by name /directors/[name] GET None A JSON object holding info about a single director, containg a name. Example:
{
name: 'Hayao Miyazaki'
}
Register as a new user /users POST A JSON object holding data about the new user, structured like:
{
name: 'Sean Kopp-Reddy',
birthday: '04/02/1992'
} 		A message confirming your registration as well as a JSON object holding data about the new user, structured like:
{
name: 'Sean Kopp-Reddy',
birthday: '04/02/1992'
}
Update user info /users/[name] PUT None A message confirming that your information has been updated.
Add movies to favorite list /users/[name]/favorites PUT A JSON object holding data about the movie to be added, structured like:
{
title: 'Fight Club',
director: 'David Fincher',
description: 'An insomniac office worker and a devil-may-care soapmaker form an underground fight club that evolves into something much, much more.'
} 		A message verifying that the movie has been added to your favorites list.
Remove movies from favorite list /users/[name]/favorites DELETE None A message verifying the movie has been removed from your favorites list.
Deregister user /users DELETE None A message verifying that your account has been deleted.
